## 前置条件

- Xray v1.3.0+
- 公网 IP
- 服务器放行 `1024-65535/UDP`

## 安装 Xray

- 使用 Arch Linux 系统
- 添加 archlinuxcn 源

```bash
sudo pacman -S xray
```
